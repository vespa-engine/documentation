## MFA reset

If you have lost your MFA information you can request it reset from Vespa.ai Support.

To reset MFA for a user, contact <a href="mailto:support@vespa.ai">support@vespa.ai</a> with the user's email address. Support will send a verification code to that email address as part of the reset process. Once received, the code must be returned to support — along with explicit confirmation that the MFA reset is authorized — before the reset can be completed.
