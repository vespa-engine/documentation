Metrics contains <code>{`requestsPerSecond`}</code> and <code>{`latencySeconds`}</code>, see <a href="https://github.com/vespa-engine/vespa/blob/master/container-disc/src/main/java/com/yahoo/container/jdisc/state/StateHandler.java" aria-label="StateHandler">StateHandler</a>. Example: <code>{`{`}</code> <code>{`&quot;time&quot;: 1661863544346,`}</code> <code>{`&quot;status&quot;: {`}</code> <code>{`&quot;code&quot;: &quot;up&quot;`}</code> <code>{`},`}</code> <code>{`&quot;metrics&quot;: {`}</code> <code>{`&quot;snapshot&quot;: {`}</code> <code>{`&quot;from&quot;: 1661863483.422,`}</code> <code>{`&quot;to&quot;: 1661863543.38`}</code> <code>{`},`}</code> <code>{`&quot;values&quot;: [`}</code> <code>{`{`}</code> <code>{`&quot;name&quot;: &quot;requestsPerSecond&quot;,`}</code> <code>{`&quot;values&quot;: {`}</code> <code>{`&quot;count&quot;: 30,`}</code> <code>{`&quot;rate&quot;: 0.5`}</code> <code>{`}`}</code> <code>{`},`}</code> <code>{`{`}</code> <code>{`&quot;name&quot;: &quot;latencySeconds&quot;,`}</code> <code>{`&quot;values&quot;: {`}</code> <code>{`&quot;average&quot;: 0.001,`}</code> <code>{`&quot;sum&quot;: 0,`}</code> <code>{`&quot;count&quot;: 0,`}</code> <code>{`&quot;last&quot;: 0.001,`}</code> <code>{`&quot;max&quot;: 0.001,`}</code> <code>{`&quot;min&quot;: 0.001,`}</code> <code>{`&quot;rate&quot;: 0`}</code> <code>{`}`}</code> <code>{`}`}</code> <code>{`]`}</code> <code>{`}`}</code> <code>{`}`}</code></td>

      <td>Service metrics</td>
      <td><code>{`/state/v1/metrics`}</code> Same as <code>{`/state/v1/health`}</code>, but with a full metrics set. A metric has a <a href="#name" aria-label="name">name</a> and <a href="#values" aria-label="values">values</a>, and can have a <a href="#description" aria-label="description">description</a> and a set of <a href="#dimensions" aria-label="dimensions">dimensions</a>: <code>{`{`}</code> <code>{`&quot;name&quot;: &quot;content.proton.documentdb.matching.rank_profile.query_setup_time&quot;,`}</code> <code>{`&quot;description&quot;: &quot;Average time (sec) spent setting up and tearing down queries&quot;,`}</code> <code>{`&quot;values&quot;: {`}</code> <code>{`&quot;average&quot;: 0,`}</code> <code>{`&quot;sum&quot;: 0,`}</code> <code>{`&quot;count&quot;: 0,`}</code> <code>{`&quot;rate&quot;: 0,`}</code> <code>{`&quot;min&quot;: 0,`}</code> <code>{`&quot;max&quot;: 0,`}</code> <code>{`&quot;last&quot;: 0`}</code> <code>{`},`}</code> <code>{`&quot;dimensions&quot;: {`}</code> <code>{`&quot;documenttype&quot;: &quot;music&quot;,`}</code> <code>{`&quot;rankProfile&quot;: &quot;default&quot;`}</code> <code>{`}`}</code> <code>{`}`}</code></td>
    </tr>

The histograms are implemented using <a href="http://hdrhistogram.org/" aria-label="HdrHistogram">HdrHistogram</a>, and the CSV result is what that library generates.</td>
