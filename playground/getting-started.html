Getting started with the Tensor Playground

This short guide explains what you can do on this site and gives a few example expressions and comments to try. When you're ready, click "Open playground" to start experimenting.

What is this?

The Tensor Playground is a small interactive editor for composing tensor expressions and comments. You add frames (either an expression or a comment), execute them and inspect the results. It uses Vespa's playground evaluation API for running expressions.

Quick actions

  • New expression: click "New expression" or press n e.
  • New comment: click "New comment" or press n c.
  • Execute: press Ctrl + Enter to execute the current frame (or all frames below it).
  • Navigation: use Up / Down (or k / j) to select frames.

Example expressions

Vector-matrix product
A = tensor(x[3])
B = tensor(x[3],y[2])
result = A * B
Try creating a frame with the above expression and execute it to inspect the result tensor.
Dense tensor dot product
a = tensor(x[2],y[2])
b = tensor(x[2],y[2])
res = a dot b

Comments

You can add inline documentation for your steps by adding a comment frame. Comments support simple markdown-like formatting.

# This is a comment about the next expression
# It will not be executed but will be shown in the flow
