Prerequisites:

  • Linux, macOS or Windows 10 Pro on x86_64 or arm64, with Podman Desktop or Docker Desktop installed, with an engine running.
    • Alternatively, start the Podman daemon: 
      $ podman machine init --memory 6000
$ podman machine start
    • See Docker Containers for system limits and other settings.
  • For CPUs older than Haswell (2013), see CPU Support.
  • Memory: Minimum {{ include.memory }} RAM dedicated to Docker/Podman. Memory recommendations.
  • Disk: Avoid NO_SPACE - the vespaengine/vespa container image + headroom for data requires disk space. Read more.
  • Homebrew to install the Vespa CLI, or download the Vespa CLI from Github releases.
