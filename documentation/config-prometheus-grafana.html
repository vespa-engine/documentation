This guide describes how to set up monitoring with Prometheus and Grafana for Cloud Vespa. This guide assumes access to a Cloud Vespa instance, as well as a Prometheus and Grafana instance. It is recommended to use Grafana Cloud, but any Grafana instance that is capable of reaching the Prometheus instance should be sufficient for the task.

Configuring Grafana

It is possible to configure Grafana both through the GUI or through configuration files. This guide covers how to configure Grafana using the configuration files and also gives a few default dashboards and configurations that is fully useable for anyone who wants to.
To start configuring Grafana, download the provisioning folder from sample-apps. This will serve as a baseline for further configuration.
